# launch

> Search and launch apps from the command line.

Search for apps on the computer and launch the selected one.

## Install

```sh
`cd path/to/launch`
```

```sh
npm link
```

## Usage

```sh
launch goo

// Google Chrome Canary.app
// Google Chrome.app
```
